# 📄 PDF Chat with RAG (Retrieval-Augmented Generation)

A lightweight AI application that allows users to interact with PDF documents using natural language.

This project demonstrates how to build a **Retrieval-Augmented Generation (RAG)** pipeline using local LLMs and vector search.

---

## 🚀 Features

- 📄 Load and process PDF documents  
- ✂️ Automatic text chunking  
- 🧬 Embedding generation  
- 🔍 Semantic search with vector database  
- 🤖 Context-aware responses using LLM  
- 💻 Runs fully locally (no API required)

---

## 🧠 Architecture

PDF → Text Extraction → Chunking → Embeddings → Vector Store → Retrieval → LLM → Answer


---

## 🛠️ Tech Stack

- Python  
- LangChain  
- FAISS (vector database)  
- Ollama (local LLM runtime)  
- Llama 3 (open-source model)
